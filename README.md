# :hammer_and_wrench: Installation Steps
1. Clone the repository

```bash
git clone git@github.com:PatrickMoraisN/pomo-timer.git
```

2. Change the working directory

```bash
cd pomo-timer
```

3. Install dependencies

```bash
npm install
```
---

4. Run

```bash
npm run dev
```

---

## Tests

You can run all tests:

```bash
npm run test
```
